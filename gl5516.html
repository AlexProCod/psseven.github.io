Подключение фоторезистора GL5516 к ESP8266

Фоторезистор GL5516 — это простой и дешевый компонент, сопротивление которого меняется в зависимости от интенсивности падающего света. Он идеально подходит для создания датчиков освещенности, систем автоматического включения уличного освещения или "умных" жалюзи. В этой статье мы рассмотрим, как правильно подключить его к ESP8266 и считывать данные.

Принцип работы

Фоторезистор — это пассивный компонент. Его сопротивление падает при увеличении освещенности и растет в темноте.

  • В темноте: Сопротивление высокое (сотни кОм – несколько МОм).
  • На свету: Сопротивление низкое (несколько кОм, для GL5516 около 5-10 кОм при ярком свете).

Поскольку ESP8266 не может измерять сопротивление напрямую, нам нужно преобразовать изменение сопротивления в изменение напряжения. Для этого используется схема делителя напряжения.

Что вам понадобится

  • Микроконтроллер ESP8266 (NodeMCU, Wemos D1 Mini).
  • Фоторезистор GL5516 (или аналог).
  • Резистор номиналом 10 кОм (для делителя напряжения).
  • Макетная плата и провода.
  • Компьютер с Arduino IDE.

Схема подключения: Делитель напряжения

Мы создадим цепь, где фоторезистор и постоянный резистор соединены последовательно. Точка их соединения подключается к аналоговому входу ESP8266.

Схема сборки:
1. Один вывод фоторезистора → 3V3.
2. Второй вывод фоторезистора → Пин A0 (и сюда же один вывод резистора 10кОм).
3. Второй вывод резистора 10кОм → GND.
Компонент Подключение Примечание
Фоторезистор (вывод 1) 3V3 Питание
Фоторезистор (вывод 2) A0 + Резистор Сигнальная точка
Резистор 10 кОм A0 ↔ GND Подтяжка к земле
💡 Почему 10 кОм? Номинал постоянного резистора должен быть примерно равен сопротивлению фоторезистора при средней освещенности. Для GL5516 оптимальным выбором является резистор от 5 кОм до 10 кОм. Это обеспечит наилучшую чувствительность в диапазоне обычного комнатного освещения.

Как это работает:

  • Когда светло: Сопротивление фоторезистора мало → напряжение на пине A0 близко к 3.3В (высокие значения ADC).
  • Когда темно: Сопротивление фоторезистора велико → напряжение на пине A0 падает к 0В (низкие значения ADC).

Примечание: Если поменять местами фоторезистор и постоянный резистор, логика изменится на противоположную (в темноте будет высокое значение, на свету — низкое).

Пример кода

Этот скетч считывает показания с аналогового входа, выводит сырое значение (0-1023) и рассчитывает примерное напряжение.

// Константы
const int photoPin = A0; // Пин подключения делителя

// Переменные
int sensorValue = 0;
float voltage = 0.0;

void setup() {
  Serial.begin(115200);
  Serial.println("Инициализация датчика освещенности...");
  delay(1000);
}

void loop() {
  // Считываем значение с аналогового входа (0-1023)
  sensorValue = analogRead(photoPin);
  
  // Переводим в напряжение (диапазон ESP8266 0-3.3В)
  voltage = sensorValue * (3.3 / 1024.0);
  
  Serial.print("Значение ADC: ");
  Serial.print(sensorValue);
  Serial.print(" | Напряжение: ");
  Serial.print(voltage, 2);
  Serial.print(" В | ");
  
  // Простая логика определения состояния
  // Пороговые значения зависят от вашего резистора и условий освещения
  if (sensorValue < 200) {
    Serial.println("Статус: Темно (Ночь)");
    // Здесь можно включить реле или светодиод
  } else if (sensorValue < 600) {
    Serial.println("Статус: Сумерки");
  } else {
    Serial.println("Статус: Светло (День)");
  }
  
  delay(500);
}

Калибровка порогов

Значения 200 и 600 в коде являются приблизительными. Для вашего проекта их нужно подобрать экспериментально:

  1. Загрузите код и откройте Serial Monitor.
  2. Посмотрите значение при полном освещении (лампа рядом).
  3. Закройте датчик рукой или выключите свет, посмотрите значение в темноте.
  4. Установите пороги в коде посередине между этими значениями.

Расширенный проект: Умный ночник

Вы можете легко превратить этот датчик в автоматический ночник, подключив светодиод или реле к цифровому пину.

  • Если sensorValue < Порог_Темноты → Включить свет (digitalWrite(HIGH)).
  • Иначе → Выключить свет.

Для плавного изменения яркости можно использовать ШИМ (PWM) на пине D1 (GPIO5), изменяя яркость светодиода пропорционально значению датчика.

Возможные проблемы

  • Показания "скачут": Фоторезисторы могут реагировать на мерцание искусственного освещения (50/60 Гц). Добавьте усреднение в коде (считать 10 значений и брать среднее) или небольшой конденсатор (0.1 мкФ) параллельно постоянному резистору.
  • Диапазон слишком узкий: Попробуйте заменить резистор 10 кОм на другой номинал (например, 5 кОм или 20 кОм), чтобы сместить рабочую точку делителя.
  • Всегда максимальное значение: Проверьте подключение. Возможно, перепутаны местами фоторезистор и постоянный резистор, либо плохой контакт с GND.

Заключение

Фоторезистор GL5516 в связке с ESP8266 — это классическое решение для задач, связанных с освещением. Несмотря на простоту, он позволяет создавать эффективные системы энергосбережения и комфорта. Главное правило — не забыть про резистор для создания делителя напряжения!
