Подключение внешнего АЦП ADS1115 к ESP8266

Встроенный аналого-цифровой преобразователь (АЦП) микроконтроллера ESP8266 имеет серьезный недостаток — низкую разрядность всего 10 бит (диапазон 0–1023) и нелинейность. Для задач, требующих высокой точности измерений (например, снятие показаний с тензодатчиков, точных датчиков тока или напряжения батарей), необходим внешний модуль. ADS1115 — это 16-битный АЦП с интерфейсом I2C, который обеспечивает высокую точность и имеет программируемый усилитель сигнала.

Зачем нужен ADS1115?

Сравним возможности встроенного АЦП ESP8266 и внешнего модуля ADS1115:

⚖️ Сравнение характеристик
Параметр ESP8266 (Встроенный) ADS1115 (Внешний)
Разрядность 10 бит (1024 уровня) 16 бит (65536 уровней)
Количество каналов 1 канал (A0) 4 канала (A0-A3)
Диапазон напряжений 0...1.0В (или 0-3.3В с делителем) Программируемый (±0.256В ... ±6.144В)
Усилитель (PGA) Нет Есть (до x16)

Вывод: ADS1115 дает в 64 раза больше градаций измерения и позволяет измерять слабые сигналы благодаря усилителю.

Что вам понадобится

  • Микроконтроллер ESP8266 (NodeMCU, Wemos D1 Mini).
  • Модуль АЦП ADS1115 (обычно на базе платы от Adafruit или аналог).
  • Соединительные провода.
  • Компьютер с Arduino IDE.

Схема подключения (I2C)

Модуль ADS1115 работает по шине I2C, что позволяет подключить его к тем же пинам, что и OLED-дисплей или другие I2C датчики.

Пин ADS1115 Пин ESP8266 Описание
VCC 3V3 (или 5V) Питание (модуль обычно имеет стабилизатор)
GND GND Общая земля
SCL D1 (GPIO5) Линия тактирования I2C
SDA D2 (GPIO4) Линия данных I2C
ADDR GND (или VCC/SDA/SCL) Выбор адреса. По умолчанию (GND) адрес 0x48

Порядок подключения:

  1. Подключите VCC и GND модуля к соответствующим пинам ESP8266.
  2. Подключите SCL к пину D1.
  3. Подключите SDA к пину D2.
  4. Пин ADDR оставьте подключенным к GND (это адрес по умолчанию 0x48). Если нужно подключить два таких модуля, второй подключите ADDR к VCC (адрес станет 0x49).
  5. Измеряемый сигнал подключайте к контактам A0, A1, A2 или A3 (и GND).
💡 Дифференциальный режим ADS1115 поддерживает не только одиночное измерение (относительно земли), но и дифференциальное (между двумя пинами, например A0 и A1). Это полезно для измерения сигналов с тензодатчиков и мостовых схем, где важно измерить разность потенциалов, игнорируя общую помеху.

Настройка Arduino IDE

Для работы с модулем потребуется библиотека от производителя чипа (Texas Instruments) или адаптированная версия.

  1. Откройте Arduino IDE.
  2. Перейдите в Sketch > Include Library > Manage Libraries.
  3. Найдите библиотеку Adafruit ADS1X15 (она универсальна для ADS1015 и ADS1115).
  4. Установите её. Библиотека Adafruit Unified Sensor установится автоматически как зависимость.

Пример кода

Этот скетч считывает показания с первого канала (A0) в одиночном режиме и выводит значение в вольтах.

#include 
#include 

// Создаем объект для ADS1115 (для ADS1015 используйте Adafruit_ADS1015)
Adafruit_ADS1115 ads;

void setup(void) {
  Serial.begin(115200);
  
  Serial.println("Инициализация ADS1115...");
  
  // Инициализация с增益 (усилением) по умолчанию (2/3x, диапазон ±6.144В)
  // Доступные значения增益:
  // ADS1015_GAIN_TWOTHIRDS  (±6.144В)
  // ADS1015_GAIN_ONE        (±4.096В)
  // ADS1015_GAIN_TWO        (±2.048В)
  // ADS1015_GAIN_FOUR       (±1.024В)
  // ADS1015_GAIN_EIGHT      (±0.512В)
  // ADS1015_GAIN_SIXTEEN    (±0.256В)
  if (!ads.begin()) {
    Serial.println("Не удалось найти ADS1115!");
    while (1);
  }
  
  Serial.println("ADS1115 найден. Начало измерений...");
}

void loop(void) {
  int16_t adc0;
  float volts;

  // Читаем значение с канала A0
  adc0 = ads.readADC_SingleEnded(0);
  
  // Переводим в напряжение
  // Для ADS1115 при усилении 2/3 (по умолчанию): 1 бит = 0.1875 мВ
  volts = adc0 * 0.0001875; 
  
  Serial.print("Канал A0: ");
  Serial.print(adc0);
  Serial.print(" | Напряжение: ");
  Serial.print(volts, 4);
  Serial.println(" В");
  
  delay(1000);
}

Настройка усиления (Gain)

Для повышения точности измерения малых сигналов (например, 0-1В) можно изменить коэффициент усиления в коде:

// Устанавливаем усиление x2 (диапазон ±2.048В) для большей точности
ads.setGain(GAIN_TWO); 
int16_t value = ads.readADC_SingleEnded(0);

Чем меньше диапазон напряжения, тем выше точность измерения (меньше размер одного бита).

Возможные проблемы

  • Не находится устройство:
    • Проверьте подключение SDA и SCL (не перепутаны ли они).
    • Убедитесь, что пин ADDR подключен правильно (к GND для адреса 0x48).
    • Запустите сканер I2C, чтобы проверить видимый адрес устройства.
  • Показания "плавают":
    • АЦП высокой чувствительности может ловить шумы. Используйте экранированные провода для измеряемого сигнала.
    • Добавьте конденсатор 0.1 мкФ между пином входа (A0) и землей.
  • Напряжение ограничено сверху:
    • Помните, что входное напряжение не должно превышать напряжение питания модуля (VCC) + 0.3В. Если измеряете напряжение выше 3.3-5В, используйте делитель напряжения.

Применение в проектах

Благодаря высокой точности и наличию 4 каналов, ADS1115 идеально подходит для:

  • Подключения тензодатчиков (весы) без использования сложных усилителей типа HX711 (хотя HX711 специализирован лучше, ADS1115 универсальнее).
  • Точного мониторинга напряжения аккумуляторных батарей.
  • Считывания данных с нескольких аналоговых датчиков одновременно (ток, свет, звук), если у ESP8266 не хватает своих пинов.
  • Измерения слабых сигналов с фотоумножителей или микрофонов.

Заключение

Модуль ADS1115 — это мощное дополнение к арсеналу ESP8266. Он решает проблему низкого разрешения встроенного АЦП и расширяет количество доступных аналоговых входов. Простота подключения по I2C и наличие готовых библиотек делают его отличным выбором для серьезных измерительных проектов.
