Подключение датчика температуры DS18B20 к ESP8266

DS18B20 — это популярный цифровой датчик температуры, работающий по протоколу 1-Wire. Его главные преимущества: водонепроницаемость (в металлическом корпусе), высокая точность (±0.5°C), возможность подключения нескольких датчиков на один пин и отсутствие необходимости в калибровке. В этой статье мы рассмотрим схему подключения и программную реализацию на ESP8266.

Особенности протокола 1-Wire

Протокол 1-Wire позволяет передавать данные и питание (при паразитном питании) по одному проводу. Однако для стабильной работы линии связи обязательно требуется подтягивающий резистор.

  • Уникальный ID: Каждый датчик имеет уникальный 64-битный серийный номер, что позволяет адресовать их индивидуально.
  • Паразитное питание: Датчик может питаться непосредственно от линии данных, но для надежной работы рекомендуется использовать внешнее питание (3 провода).
  • Масштабируемость: На одну шину можно повесить десятки датчиков.

Что вам понадобится

  • Микроконтроллер ESP8266 (NodeMCU, Wemos D1 Mini).
  • Датчик температуры DS18B20 (в корпусе или голый модуль).
  • Резистор номиналом 4.7 кОм (обязательно!).
  • Соединительные провода.
  • Компьютер с Arduino IDE.

Схема подключения

Для подключения нам понадобятся три контакта датчика: GND, VCC и DATA. Самый важный момент — установка подтягивающего резистора между линиями питания и данных.

Пин DS18B20 Пин ESP8266 Описание
GND (Черный) GND Общая земля
VCC (Красный) 3V3 Питание 3.3 В
DATA (Желтый/Белый) D4 (GPIO2) или любой GPIO Линия данных 1-Wire
⚡
❗ Критически важно: Подтягивающий резистор
Между линией DATA и линией VCC необходимо подключить резистор номиналом 4.7 кОм (допустимо от 3.3 кОм до 10 кОм). Без этого резистора связь с датчиком будет нестабильной или отсутствовать вовсе.

Порядок сборки:

  1. Подключите GND датчика к GND ESP8266.
  2. Подключите VCC датчика к 3V3 ESP8266.
  3. Подключите DATA датчика к выбранному GPIO (например, D4).
  4. Припаяйте или вставьте в макетную плату резистор 4.7 кОм, соединив ножки DATA и VCC.

Настройка Arduino IDE

Для работы с DS18B20 потребуются две библиотеки:

  1. OneWire — реализация протокола 1-Wire.
  2. DallasTemperature — упрощает работу с датчиками серии DS18x20.

Установите их через меню: Sketch > Include Library > Manage Libraries, найдите по названию и нажмите Install.

Пример кода

Этот скетч находит все подключенные датчики на шине, считывает температуру и выводит её в последовательный порт.

#include 
#include 

// Пин подключения датчика
#define ONE_WIRE_BUS D4 

// Создаем объект OneWire
OneWire oneWire(ONE_WIRE_BUS);

// Передаем ссылку на OneWire в библиотеку DallasTemperature
DallasTemperature sensors(&oneWire);

void setup(void) {
  Serial.begin(115200);
  Serial.println("Инициализация датчика DS18B20...");
  
  // Запуск библиотеки
  sensors.begin();
  
  // Вывод количества найденных датчиков
  Serial.print("Найдено датчиков: ");
  Serial.println(sensors.getDeviceCount());
}

void loop(void) { 
  // Запрос температуры всем устройствам на шине
  sensors.requestTemperatures(); 
  
  // Получаем температуру первого датчика (индекс 0)
  float tempC = sensors.getTempCByIndex(0);
  
  // Проверка на ошибку чтения
  if (tempC == -127.00) {
    Serial.println("Ошибка чтения датчика! Проверьте подключение.");
  } else {
    Serial.print("Температура: ");
    Serial.print(tempC);
    Serial.println(" °C");
    
    // Если нужно в Фаренгейтах:
    // Serial.print(sensors.toFahrenheit(tempC));
    // Serial.println(" °F");
  }
  
  delay(2000); // Пауза 2 секунды
}
💡 Работа с несколькими датчиками Если у вас подключено несколько датчиков к одному пину, вы можете перебирать их в цикле:
for (int i = 0; i < sensors.getDeviceCount(); i++) { ... getTempCByIndex(i) ... }
Или обращаться к ним по уникальному адресу (Address), используя функцию getAddress().

Возможные проблемы

  • -127.00 °C: Эта ошибка означает, что библиотека не смогла прочитать данные. Чаще всего причина в отсутствии подтягивающего резистора 4.7 кОм или плохом контакте проводов.
  • Нестабильные показания: Попробуйте экранировать длинные провода или уменьшить их длину. Убедитесь, что питание стабильно.
  • Конфликт пинов: Избегайте использования пина D0 (GPIO16) для 1-Wire, так как он не поддерживает все необходимые режимы прерываний в некоторых библиотеках. Лучше использовать D1-D8.

Применение в умном доме

Благодаря герметичному корпусу, DS18B20 идеально подходит для:

  • Мониторинга температуры воды в аквариуме или системе отопления.
  • Измерения температуры почвы в умной теплице.
  • Контроля температуры в холодильнике или погребе.
  • Уличных метеостанций (провода нужно дополнительно изолировать).

Заключение

DS18B20 — это надежный и недорогой стандарт для измерения температуры в проектах на ESP8266. Главное правило успешного подключения — не забыть про резистор 4.7 кОм. После правильной сборки датчик будет служить годами без дополнительной калибровки.
