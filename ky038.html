Подключение звукового датчика KY-038 к ESP8266

Модуль KY-038 — это простой и дешевый датчик звука на основе электретного микрофона. Он позволяет обнаруживать наличие звука, хлопки или измерять уровень громкости. Модуль имеет два типа выхода: аналоговый (для измерения уровня сигнала) и цифровой (для детектирования превышения порога). В этой статье мы рассмотрим оба режима работы.

Особенности модуля KY-038

Модуль построен на базе компаратора LM393 и электретного микрофона. Его ключевые особенности:

  • Два выхода: Аналоговый (AO) выдает напряжение, пропорциональное громкости; Цифровой (DO) срабатывает как переключатель.
  • Регулировка чувствительности: Синий потенциометр на плате позволяет настроить порог срабатывания цифрового выхода.
  • Индикация: Светодиод на плате загорается при срабатывании цифрового выхода.
  • Напряжение питания: Работает от 3.3В до 5В, что идеально совместимо с ESP8266.

📈 Аналоговый режим (AO)

Выдает изменяющееся напряжение в зависимости от громкости звука. Используется для измерения уровня шума, создания визуализаторов или детектирования силы хлопка.

🔘 Цифровой режим (DO)

Выдает логический 0 или 1. Срабатывает, когда звук превышает порог, заданный потенциометром. Идеален для простых задач: "хлопни в ладоши — включи свет".

Что вам понадобится

  • Микроконтроллер ESP8266 (NodeMCU, Wemos D1 Mini).
  • Звуковой датчик KY-038.
  • Соединительные провода.
  • Компьютер с Arduino IDE.
  • USB-кабель.

Схема подключения

В зависимости от задачи, вы можете использовать только один выход или оба сразу. Ниже приведена таблица для полного подключения.

Пин KY-038 Пин ESP8266 Описание
- (GND) GND Общая земля
+ (VCC) 3V3 (или 5V) Питание
AO A0 Аналоговый выход (уровень звука)
DO D1 (GPIO5) Цифровой выход (пороговое значение)

Порядок подключения:

  1. Подключите GND модуля к GND ESP8266.
  2. Подключите VCC модуля к 3V3 ESP8266.
  3. Для аналогового режима: подключите AO к пину A0.
  4. Для цифрового режима: подключите DO к любому свободному GPIO, например D1.
💡 Настройка чувствительности Перед загрузкой кода настройте синий потенциометр на плате датчика:
1. Подайте питание на модуль.
2. Создайте тишину или фоновый шум.
3. Медленно вращайте отверткой потенциометр, пока светодиод на модуле не погаснет (или не загорится, в зависимости от логики).
4. Проверьте срабатывание хлопком. Светодиод должен вспыхивать только при громком звуке.

Пример кода: Цифровой режим (Хлопок)

Этот скетч использует цифровой выход. При обнаружении громкого звука (превышение порога) загорается встроенный светодиод ESP8266.

// Пины
const int digitalPin = D1; // Пин DO датчика
const int ledPin = LED_BUILTIN; // Встроенный светодиод

// Переменные
int sensorState = 0;
int lastSensorState = 0;

void setup() {
  Serial.begin(115200);
  pinMode(digitalPin, INPUT);
  pinMode(ledPin, OUTPUT);
  
  digitalWrite(ledPin, HIGH); // Выключаем светодиод (инверсная логика)
  
  Serial.println("Режим: Детектор хлопка (Цифровой)");
  delay(1000);
}

void loop() {
  sensorState = digitalRead(digitalPin);
  
  // Логика датчика: обычно LOW при звуке, HIGH в тишине
  // Проверьте ваш модуль, возможно логика инверсна
  if (sensorState == LOW) { 
    if (lastSensorState == HIGH) {
      Serial.println(">>> ХЛОПОК ОБНАРУЖЕН! <<<");
      digitalWrite(ledPin, LOW); // Включаем светодиод
    }
  } else {
    if (lastSensorState == LOW) {
      digitalWrite(ledPin, HIGH); // Выключаем светодиод
    }
  }
  
  lastSensorState = sensorState;
  delay(100); // Небольшая задержка для стабильности
}

Пример кода: Аналоговый режим (Уровень шума)

Этот скетч считывает аналоговое значение и выводит уровень шума в реальном времени. Полезно для создания индикаторов громкости.

// Пины
const int analogPin = A0; // Пин AO датчика

// Переменные
int sensorValue = 0;
int noiseLevel = 0;

void setup() {
  Serial.begin(115200);
  Serial.println("Режим: Монитор уровня шума (Аналоговый)");
  delay(1000);
}

void loop() {
  // Считываем сырое значение (0-1023)
  sensorValue = analogRead(analogPin);
  
  // В тишине значение будет около некоторой базы (например, 300-400)
  // При звуке значения будут скакать вверх
  
  Serial.print("Уровень сигнала: ");
  Serial.print(sensorValue);
  
  // Простая индикация статуса
  if (sensorValue > 600) {
    Serial.println(" - ГРОМКО!");
  } else if (sensorValue > 450) {
    Serial.println(" - Средний шум");
  } else {
    Serial.println(" - Тишина");
  }
  
  delay(100);
}
⚠️ Особенности аналогового сигнала Микрофон выдает быстро меняющийся сигнал. В тишине показания не будут равны 0, они будут колебаться вокруг некоторого среднего значения (смещения). Для точного анализа звука (например, вычисления среднеквадратичного значения) нужно делать серию быстрых замеров и математически обрабатывать их.

Возможные проблемы

  • Ложные срабатывания: Если датчик реагирует на малейший шум, уменьшите чувствительность потенциометром.
  • Нет реакции на хлопок: Увеличьте чувствительность. Убедитесь, что вы используете правильный пин (DO для цифрового, AO для аналогового).
  • Нестабильные показания в аналоговом режиме: Это нормально для микрофонов. Используйте усреднение значений в коде, если нужно получить стабильный уровень фона.

Применение в проектах

  • Управление светом по хлопку: Классический проект для умного дома.
  • Датчик шума: Отправка уведомлений, если в комнате слишком шумно (актуально для детской или офиса).
  • Охранная система: Реагирование на звук разбития стекла (требуется более сложная обработка сигнала).
  • Голосовой активатор: Пробуждение устройства по голосовой команде (в упрощенном варианте).

Заключение

Датчик KY-038 — отличный выбор для начала работы со звуком в проектах на ESP8266. Он прост в подключении, дешев и имеет гибкую настройку чувствительности. Для сложных задач распознавания речи лучше использовать специализированные модули, но для детектирования наличия звука или хлопков KY-038 подходит идеально.
