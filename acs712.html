Подключение датчика тока ACS712 к ESP8266

Датчик тока ACS712 — это популярный модуль на основе эффекта Холла, позволяющий измерять силу тока в цепях переменного (AC) и постоянного (DC) напряжения. В этой статье мы рассмотрим, как подключить ACS712 к микроконтроллеру ESP8266, рассчитать реальное значение тока и вывести данные в_serial монитор.

Особенности модуля ACS712

Модуль ACS712 бывает трех типов в зависимости от диапазона измеряемого тока:

  • ACS712ELCTR-05B: диапазон ±5А (чувствительность 185 мВ/А).
  • ACS712ELCTR-20A: диапазон ±20А (чувствительность 100 мВ/А).
  • ACS712ELCTR-30A: диапазон ±30А (чувствительность 66 мВ/А).

Принцип работы основан на измерении падения напряжения на выходе датчика, которое пропорционально протекающему току. При отсутствии тока напряжение на выходе равно половине напряжения питания (VCC / 2).

Что вам понадобится

  • Микроконтроллер ESP8266 (NodeMCU, Wemos D1 Mini).
  • Датчик тока ACS712 (любого диапазона).
  • Соединительные провода.
  • Нагрузка для тестирования (лампочка, мотор и т.д.).
  • Источник питания нагрузки.
  • Компьютер с Arduino IDE.

Схема подключения

Датчик ACS712 имеет три контакта для подключения к микроконтроллеру и два винтовых клеммника для подключения измеряемой цепи.

Пин ACS712 Пин ESP8266 Описание
VCC 3V3 (или 5V) Питание датчика (работает от 5В, но выход совместим с 3.3В логикой)
GND GND Общая земля
OUT A0 (GPIO17) Аналоговый выход сигнала
⚠️ ТЕХНИКА БЕЗОПАСНОСТИ Будьте предельно осторожны при измерении тока в цепях 220В.
1. Все соединения силовой цепи должны быть надежно изолированы.
2. Не прикасайтесь к оголенным частям цепи под напряжением.
3. Рекомендуется сначала протестировать схему на безопасном напряжении (например, 12В DC).

Порядок подключения:

  1. Подключите VCC датчика к пину 3V3 (или 5V, если есть внешний источник) на ESP8266.
  2. Подключите GND датчика к GND ESP8266.
  3. Подключите вывод OUT датчика к аналоговому входу A0 на ESP8266.
  4. Разомкните цепь нагрузки и включите её разрыв в винтовые клеммы датчика (контакты IP+ и IP-). Полярность для AC не важна, для DC влияет на знак тока.

Принцип расчета тока

ESP8266 имеет 10-битный АЦП (диапазон значений 0–1023), который преобразует напряжение 0–1.0В (на большинстве плат NodeMCU стоит делитель, поэтому фактический диапазон 0–3.3В соответствует 0–1023).

Формула расчета:
1. Напряжение на пине: U = (ADC_Value / 1024.0) * 3.3
2. Смещение нуля: U_zero = 2.5В (при питании 5В) или ~1.65В (при питании 3.3В)
3. Ток: I = (U - U_zero) / Чувствительность

Так как ESP8266 работает с 3.3В логикой, а датчик часто питают от 5В для стабильности, напряжение покоя (без тока) будет около 2.5В. Однако вход A0 у ESP8266 ограничен 1.0В (на старых платах) или 3.3В (на новых с делителем). Для упрощения в примере ниже мы будем считать, что плата имеет встроенный делитель напряжения на входе A0 (как NodeMCU v3).

Пример кода

Следующий скетч считывает показания с аналогового входа, усредняет их для стабильности и переводит в Амперы.

// Константы для калибровки
const int sensorPin = A0;       // Пин подключения датчика
const float zeroOffset = 512.0; // Значение АЦП без тока (середина диапазона 1024)
                                // Требуется калибровка! Измерьте значение без нагрузки.
const float sensitivity = 100.0; // Чувствительность в мВ/А (100 для 20А, 185 для 5А)
                                // Для ESP8266 с делителем нужно пересчитать коэффициент

// Переменные
float current = 0.0;
int adcValue = 0;

void setup() {
  Serial.begin(115200);
  Serial.println("Инициализация датчика тока ACS712...");
  delay(1000);
}

void loop() {
  // Считываем несколько значений для усреднения (фильтрация шумов)
  long sum = 0;
  for (int i = 0; i < 100; i++) {
    sum += analogRead(sensorPin);
    delay(5);
  }
  
  float avgADC = sum / 100.0;
  
  // Расчет напряжения (для ESP8266 с диапазоном 0-3.3В и 10 бит)
  float voltage = avgADC * (3.3 / 1024.0);
  
  // Расчет тока
  // Напряжение нулевой точки при питании 5В обычно 2.5В
  // Но если питаем от 3.3В, то ноль будет около 1.65В
  // В данном примере предполагаем питание датчика от 3.3В для простоты с ESP8266
  float zeroVoltage = 1.65; 
  
  float deltaV = voltage - zeroVoltage;
  
  // Чувствительность для 20А модуля = 0.100 В/А
  // Для 5А модуля = 0.185 В/А
  current = deltaV / (sensitivity / 1000.0);
  
  Serial.print("ADC: ");
  Serial.print(avgADC);
  Serial.print(" | Напряжение: ");
  Serial.print(voltage, 3);
  Serial.print(" В | Ток: ");
  Serial.print(current, 3);
  Serial.println(" А");
  
  delay(500);
}
💡 ВАЖНО: Калибровка Значение zeroOffset (или напряжение нуля) может отличаться в зависимости от конкретного экземпляра датчика и качества питания.
1. Загрузите код без подключенной нагрузки.
2. Посмотрите в Serial Monitor значение тока.
3. Если оно не равно 0.00, подкорректируйте переменную zeroVoltage или zeroOffset в коде, пока ток не станет нулевым.

Тестирование и нюансы

  • Шумы: Аналоговый сигнал чувствителен к помехам. Используйте усреднение (как в коде выше) или конденсатор небольшой емкости между OUT и GND.
  • Питание: Для точных измерений лучше питать датчик от стабильных 5В, а сигнал на A0 подавать через делитель, если ваша плата ESP8266 не имеет встроенного делителя на входе A0.
  • Переменный ток: Для измерения AC тока нужно снимать показания быстро в течение периода синусоиды, находить максимальное и минимальное значение, либо вычислять среднеквадратичное значение (RMS).

Заключение

Датчик ACS712 — отличное решение для мониторинга энергопотребления в проектах умного дома. Вы можете отслеживать расход электроэнергии лампами, приборами или контролировать зарядку аккумуляторов. Комбинируя этот датчик с Wi-Fi возможностями ESP8266, можно отправлять данные о токе на сервер или в Telegram.
